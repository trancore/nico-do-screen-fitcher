# NicoDoScreenFitcher

## このアプリケーションについて

このツールは、ニコニコ動画の動画を横幅いっぱいに表示するための Chrome 拡張機能です。

トグルボタンだけで、ON と OFF の切り替えのみができます。

![popup](images/popup.png)

## 📡 使用ライブラリ・フレームワーク

| ライブラリ・フレームワーク | バージョン | 使用意図                 |
| -------------------------- | ---------- | ------------------------ |
| Chrome extention manifest  | v3         | 拡張機能プラットフォーム |
